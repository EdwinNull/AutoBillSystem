资源文件目录

请将以下文件放置在此目录中：

必需文件：
- icon.ico: 应用程序图标文件 (推荐尺寸: 256x256)
  * 用于exe文件的图标显示
  * 支持ICO格式，包含多种尺寸
  * 可以使用在线工具将PNG转换为ICO

可选文件：
- splash.png: 启动画面图片
  * 程序启动时显示的欢迎界面
  * 推荐尺寸: 400x300 或 600x400
  * PNG格式，支持透明背景

- logo.png: 应用程序标志
  * 在程序界面中显示的标志
  * 推荐尺寸: 128x128 或 64x64
  * PNG格式，支持透明背景

- banner.png: 横幅图片
  * 用于关于对话框或主界面顶部
  * 推荐尺寸: 400x100
  * PNG格式

图标制作建议：
1. 使用专业的图标设计软件（如Adobe Illustrator、Figma）
2. 确保图标在小尺寸下仍然清晰可辨
3. 使用简洁的设计，避免过于复杂的细节
4. 选择与汽车维修主题相关的元素（如扳手、齿轮、汽车等）
5. 保持与系统整体UI风格的一致性

在线图标资源：
- Flaticon (https://www.flaticon.com/)
- Icons8 (https://icons8.com/)
- Iconfinder (https://www.iconfinder.com/)

ICO转换工具：
- ConvertICO (https://convertico.com/)
- ICO Convert (https://icoconvert.com/)
- 在线ICO转换器 (https://www.aconvert.com/cn/icon/)

注意事项：
- 确保使用的图标具有适当的使用许可
- 商业使用时请购买相应的许可证
- 建议创建多个尺寸的图标以适应不同显示需求
